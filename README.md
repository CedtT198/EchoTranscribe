# 📝 Projet : Générateur automatique de résumés de réunions

## 🚀 Technologies utilisées

- 🌐 **Frontend** : React
- ⚙️ **Backend** : Spring Boot  
- 🗄️ **Base de données** : MongoDB
- ⚡ **Cache et performance** : Redis  
<!-- - 📝 **Conception** : Looping   -->
- 📊 **Diagrammes de flux** : Swimlane sur Draw.io


---

## 🎯 Objectif du projet

Développer une **application web intelligente** permettant de transformer des contenus bruts (audio, vidéo ou texte) en **résumés clairs, concis et exploitables**.  

L’application couvre plusieurs fonctionnalités :

- 📂 **Uploader** un fichier audio/vidéo (ex. enregistrement de réunion) ou du texte (chat, compte rendu brut).  
- 🧠 **IA de transcription** : 
    - extraction audio d'une vidéo.    
    - audio → texte.    
- ⚙️ **Options de personnalisation** des résultats :
  - Résumé **ultra-concis** (quelques phrases clés).  
  - Résumé **détaillé** (plusieurs points explicatifs).  
  - Résumé **orienté décisions** (mise en avant des choix et actions à mener).  
- ✨ **Génération automatique d’un résumé** sous forme de points clés.  
- 🌍 **Support multilingue** (transcription et résumé).  
- 📤 **Export multi-supports** :
  - Fichier **PDF**  
  - Fichier **Word**  
  - Envoi par **email**

## 💡 Valeur ajoutée pour l’entreprise

- ⏳ **Gain de temps** : fini la rédaction manuelle de comptes rendus.  
- 🗂️ **Centralisation** : stockage et historisation des réunions.  
- 📊 **Analyse stratégique** : suivi des décisions et projets.  
- 🔄 **Polyvalence** : utilisable par divers départements :

  - Équipes IT  
  - Support client, etc.
